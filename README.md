## All in One Study and Working App

This app helps you for focus your work and keeps everthing one screen.

## Features

- [x] Timer
- [x] Timer Alert
- [x] Todo List
- [x] Notes List
- [x] Light and Dark Mode

## What i use and learn

- [x] React
- [x] React Router
- [x] React Hooks
- [x] Redux
- [x] Local Storage
- [x] React Icons Kit
- [x] Tailwind CSS
- [x] Material UI
- [x] Responsive Design
